Tutorials
===============

Welcome to the Tutorial Guide. This guide offers step-by-step instructions for using ``barotropy`` to generate barotropic fluid models.
The tutorials focus on the two main aspects of the software: two-component systems (e.g., water-air) and single-component systems (e.g., water-steam).
Additionally, an overview of the various configuration file options is included to help understnad the software capabilities
